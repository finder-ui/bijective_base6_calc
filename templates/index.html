A Primer on Numerical Systems

Most number systems we use are positional. This means the value of a digit depends on its position within a number. The "base" of a system tells us how many unique digits are used. The value of each position is an exponent of the base, starting from 0 on the right.

In these standard systems, the digit zero is essential. It is not just a number, but a fundamental placeholder. It signifies an empty position, like the tens place in 101. The bijective system you are exploring is fascinating because it works without this placeholder.

Understanding the Bijective System

This system is a bijective base-6 numeration. It is positional, but with two key differences: it has no zero, and its digits run from 1 to 6. Because there is no zero to act as a placeholder, the system "rolls over" differently. After 6, the next number is 11.

The Conversion Algorithm Explained

The trick to converting from decimal to bijective base-6 is to temporarily shift the number system. Computers naturally work with remainders from 0 to 5, but our digits are 1 to 6. We bridge this gap by subtracting 1 before each calculation.
