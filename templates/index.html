A Primer on Numerical Systems

Most number systems we use are positional. This means the value of a digit depends on its position within a number. The "base" of a system tells us how many unique digits are used. The value of each position is an exponent of the base, starting from 0 on the right.

  • Base-10 (Decimal): We use ten digits (0-9). In 345, the value is (3 * 10¬≤) + (4 * 10¬π) + (5 * 10‚Å∞) = 345.
  • Base-2 (Binary): Computers use two digits (0, 1). In 1101, the value is (1 * 2¬≥) + (1 * 2¬≤) + (0 * 2¬π) + (1 * 2‚Å∞) = 13.

In these standard systems, zero is an essential placeholder. The bijective system is fascinating because it works without it.

Understanding the Bijective System

This system is a bijective base-6 numeration. It is positional, but it has no zero; the digits are 1, 2, 3, 4, 5, 6. Because there is no zero, the system "rolls over" differently. After 6, the next number is 11.

The Conversion Algorithm Explained

The trick to converting from decimal is to temporarily shift the number system. Computers use remainders from 0-5, but our digits are 1-6. We bridge this by subtracting 1 before each calculation. Let's convert decimal 43:

  1. Step 1: (43 - 1) % 6 = 0. The 0th digit in "123456" is '1'. New number is (43 - 1) / 6 = 7.
  2. Step 2: (7 - 1) % 6 = 0. The 0th digit is '1'. New number is (7 - 1) / 6 = 1.
  3. Step 3: (1 - 1) % 6 = 0. The 0th digit is '1'. New number is (1 - 1) / 6 = 0.

Reversing the digits gives 111. Thus, 43 in decimal is 111 in bijective base-6.

Practical Applications

  • Spreadsheet Columns: Excel and Google Sheets use a bijective base-26 system (A-Z). After Z comes AA, then AB. This is a perfect real-world example where there is no "zeroth" column.
  • URL Shorteners: Services like bit.ly use a bijective base-62 system (a-z, A-Z, 0-9) to map massive database IDs to short, unique, and readable strings.

Deeper Mathematical Connections

The properties of bijective numeration extend into more complex areas of computer science and mathematics, such as cryptography and algorithmic enumeration, providing a unique and powerful tool for representing data and sequences.
